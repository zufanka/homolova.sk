Methodology & data sources

Scope. Only recipients registered in Alpine Convention 2024 perimeter (1,010 AT Gemeinden + 1,648 IT comuni). All recipient types included; anonymised excluded.

Schemes vs. categories. The row-detail drawer shows the same money at two levels of detail. Schemes (Annex IX code & name) are the raw line items as they appear in the transparency files — the specific EU-regulation measure code each payment was made under (e.g. I.1 BISS, VI.18 ANC, IV.5 sectoral fruit & veg). Money by category rolls those scheme codes up into ~20 broader, human-readable buckets (BISS, CRISS, eco-schemes, agri-environment, ANC, investment, sectoral, etc.) so AT and IT payments can be compared on a common axis. The schemes view is the audit trail back to the source data; the category view is the cross-country comparison view.

Mountain class. Municipalities are classified as T (totally mountainous), P (partially mountainous) or NM (non-mountain) using the Allegato B (Law 991/1952) surface-share rule: T if ≥ 80 % of the territory is above 600 m, P if ≥ 33 % (or ≥ 10 % with the administrative centre itself above 600 m), NM otherwise. Italy uses ISTAT's 2021 altimetric tables. Austria has no equivalent national table, so we derive the same statistics ourselves from the Copernicus GLO-90 DEM (90 m resolution) run over the AT Gemeinde polygons of the Alpine Convention layer.

Sources. AMA Transparenzdatenbank for Austria; AGEA Trasparenza CSV for Italy; ISTAT altimetric tables (Dec 2021) for IT; Copernicus GLO-90 DEM for AT (via the public AWS bucket).
