Methodology & data sources

Scope. Only recipients registered in Alpine Convention 2024 perimeter (1,010 AT Gemeinden + 1,648 IT comuni). Natural persons and anonymised small recipients excluded. Pure administrative public bodies (ministries, comuni, regions, universities, churches, mountain communities) excluded.

Alpine-pasture flag (🐄). No central registry exists, so the flag is inferred from two signals. (1) Name match. The recipient name is matched against a regex chain for nine known collective alm forms — Almgemeinschaft, Agrargemeinschaft, Interessentschaft, Eigenverwaltung B.N.R., ASUC, Regola, Magnifica Comunità, Comunella, Malga. (2) Austria-only payment-profile fallback. An Austrian recipient is also flagged if its CAP money looks like an alpine farm even when the name didn't match: BISS > 0, ANC (mountain compensation) > 0, agri-environment > 0, and (ANC + agri-env) > 3 × BISS. The intuition: alpine farms get tiny BISS (low €/ha rate × little area) but big ANC and ÖPUL top-ups, so that ratio is a fingerprint. Italy uses only signal (1), because IT data doesn't separate ANC from BISS as cleanly. The flag therefore leans toward false negatives — e.g., a Soc. Agr. … S.r.l. running an alpine pasture would be classified company, not alm.

Schemes vs. categories. The row-detail drawer shows the same money at two levels of detail. Schemes (Annex IX code & name) are the raw line items as they appear in the transparency files — the specific EU-regulation measure code each payment was made under (e.g. I.1 BISS, VI.18 ANC, IV.5 sectoral fruit & veg). Money by category rolls those scheme codes up into ~20 broader, human-readable buckets (BISS, CRISS, eco-schemes, agri-environment, ANC, investment, sectoral, etc.) so AT and IT payments can be compared on a common axis. The schemes view is the audit trail back to the source data; the category view is the cross-country comparison view.

Sources. AMA Transparenzdatenbank for Austria; AGEA Trasparenza CSV for Italy; ISTAT altimetric tables (Dec 2021) for the IT mountain class T / P / NM derived from the Allegato B (Law 991/1952) rule. Per-ha rates from the BMLUK and EU Commission's Austria CAP Strategic Plan factsheet (Sept 2025) and MASAF / Pianeta PSR.
